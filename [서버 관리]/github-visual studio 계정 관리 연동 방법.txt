[1] 깃허브 repository 생성
	1. 깃허브 로그인 (tippy.hosting, inori.archive)
	2. New repository
	3. Repository name 입력
	4. Add a README file 선택
	5. Setting-Pages-Branch main 선택 후 저장
	
[2] Git PC 설정
	1. Git 설치 (Git Bash)
	2. Git Bash 에서 git config --global user.name "이름"
	3. Git Bash 에서 git config --global user.email "이메일"
	4. Git Bash 에서 git config --list 로 설정 확인

[3] Visual Studio 설정
	1. Visual Studio 내 깃허브 로그인
	2. 구글 크롬 이용 해당 계정 GitHub 연동
	3. 파일이 있을 경우 - Open Folder로 해당 파일 열기
	4-1. 파일이 없을 경우 - 메인화면 진입 후 Clone Git Repository
	4-2. 해당 GitHub 주소 입력 후 파일 위치 선택(바로설치위치)

[!] 다른계정으로 Visual Studio 선택시
	1. Git Bash 에서 git config --global user.name, email 재설정
	2. 자격 증명 관리자-windows 자격 증명-github 제거
	3. Visual Studio 로그아웃
	4. 해당 계정 크롬 선택 및 Visual Studio 재로그인
